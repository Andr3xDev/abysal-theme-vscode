## Match with System

<!--eslint-skip-->

```jsonc
// .vscode/settings.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredDarkColorTheme": "Abysal Obsidian",
  "workbench.preferredLightColorTheme": "Abysal Marble"
}
```
